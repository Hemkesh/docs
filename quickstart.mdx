## Get your API key

<Steps>
  <Step title="Log into Roulette">
    Go to [useroulette.com](https://www.useroulette.com/home) and sign in to your account.
  </Step>
  <Step title="Navigate to Settings">
    Click on your profile menu and select **Settings** â†’ **API Keys**.
  </Step>
  <Step title="Generate a new key">
    Click **Create API Key**, give it a name (e.g., "Development"), and copy the generated key.

    <Warning>Store your API key securely. It won't be shown again!</Warning>
  </Step>
</Steps>

## Make your first request

Let's list the companies in your pipeline:

<CodeGroup>

```bash cURL
curl -X GET "https://useroulette.com/api/v1/companies" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://useroulette.com/api/v1/companies', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    'https://useroulette.com/api/v1/companies',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    }
)

print(response.json())
```

</CodeGroup>

## Create a company

Now let's add a new company to your pipeline:

<CodeGroup>

```bash cURL
curl -X POST "https://useroulette.com/api/v1/companies" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "account_id": "YOUR_ACCOUNT_ID",
    "name": "TechStartup Inc",
    "website": "https://techstartup.io",
    "description": "Next-gen developer tools",
    "source": "referral"
  }'
```

```javascript JavaScript
const response = await fetch('https://useroulette.com/api/v1/companies', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    account_id: 'YOUR_ACCOUNT_ID',
    name: 'TechStartup Inc',
    website: 'https://techstartup.io',
    description: 'Next-gen developer tools',
    source: 'referral'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'https://useroulette.com/api/v1/companies',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'account_id': 'YOUR_ACCOUNT_ID',
        'name': 'TechStartup Inc',
        'website': 'https://techstartup.io',
        'description': 'Next-gen developer tools',
        'source': 'referral'
    }
)

print(response.json())
```

</CodeGroup>

<Tip>
  You can find your `account_id` in the URL when viewing your dashboard:
  `useroulette.com/home/{account_id}/...`
</Tip>
