## Get your API key

<Steps>
  <Step title="Log into Roulette">
    Go to [useroulette.com](https://www.useroulette.com/home) and sign in to your account.
  </Step>
  <Step title="Navigate to Settings">
    Click on your profile menu and select **Settings** â†’ **API Keys**.
  </Step>
  <Step title="Generate a new key">
    Click **Create API Key**, give it a name (e.g., "Development"), and copy the generated key.

    <Warning>Store your API key securely. It won't be shown again!</Warning>
  </Step>
</Steps>

## Make your first request

Let's list the companies in your pipeline:

<CodeGroup>

```bash cURL
curl -X GET "https://useroulette.com/api/v1/companies" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://useroulette.com/api/v1/companies', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    'https://useroulette.com/api/v1/companies',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    }
)

print(response.json())
```

</CodeGroup>

## Create a company

The recommended way to create companies is using the intelligent AI upload endpoint.
It can process emails, form submissions, or any text containing pitch deck URLs:

<CodeGroup>

```bash cURL
curl -X POST "https://useroulette.com/api/v1/companies/upload/ai" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "account_id=YOUR_ACCOUNT_ID" \
  -F "text=From: founder@startup.com
Subject: TechStartup Series A

Hi, here's our pitch deck: https://docsend.com/view/example123

We're raising a $5M Series A." \
  -F "name=TechStartup Inc"
```

```javascript JavaScript
const formData = new FormData();
formData.append('account_id', 'YOUR_ACCOUNT_ID');
formData.append('text', `From: founder@startup.com
Subject: TechStartup Series A

Hi, here's our pitch deck: https://docsend.com/view/example123

We're raising a $5M Series A.`);
formData.append('name', 'TechStartup Inc');

const response = await fetch('https://useroulette.com/api/v1/companies/upload/ai', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: formData
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'https://useroulette.com/api/v1/companies/upload/ai',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY'
    },
    data={
        'account_id': 'YOUR_ACCOUNT_ID',
        'text': '''From: founder@startup.com
Subject: TechStartup Series A

Hi, here's our pitch deck: https://docsend.com/view/example123

We're raising a $5M Series A.''',
        'name': 'TechStartup Inc'
    }
)

print(response.json())
```

</CodeGroup>

The AI endpoint automatically:
- Extracts pitch deck URLs from the text
- Downloads and stores the deck
- Detects the company source (email, form, referral, etc.)
- Suggests an appropriate pipeline status

<Tip>
  You can find your `account_id` in the URL when viewing your dashboard:
  `useroulette.com/home/{account_id}/...`
</Tip>
